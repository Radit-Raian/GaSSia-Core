# GaSSia-Core

**Galaxy Spectra & Type Ia Supernova Analysis and Correlation!!**

## Overview

**GaSSia-Core** is a modular pipeline for analyzing galaxy spectra and investigating correlations between host galaxy spectral properties and Type Ia supernova characteristics. The pipeline integrates tools from the Python scientific ecosystem, including:

- `Astropy`
- `NumPy`
- `Matplotlib`
- `Pandas`
-  `ppxf`

It is designed to:

- Load and preprocess spectroscopic data (e.g., FITS spectra)
- Perform background handling and normalization of spectra
- Identify key emission and absorption lines
- Extract spectral line measurements (fluxes, equivalent widths, etc.)
- Visualize spectra and diagnostic features
- Support exploratory analysis of galaxy–supernova connections

This pipeline is part of my work in astronomical spectroscopy, particularly focusing on understanding how host galaxy spectral properties relate to Type Ia supernova behavior. By analyzing spectral line features such as Hα, Hβ, [O III], and [N II], the project aims to build a foundation for studying environmental dependencies of Type Ia supernovae and their implications in astrophysics and cosmology.
